# Domino's Pakistan Clone

A React clone of the Domino's Pizza Pakistan website (dominos.com.pk), inspired by their menu and Ramadan deals page.

## Features

- **Header** – Logo, navigation (Menu, Deals, Ramadan Deals, Track Order), and Order Now button
- **Hero section** – Gradient banner with CTA buttons
- **Deals banner** – BOGO, combo, and delivery offers
- **Ramadan deals** – Family Deal, Iftar Special, Suhoor Combo cards
- **Menu section** – Pizzas (Chicken Tikka, Pepperoni, Fiery Chicken, etc.) and Sides (Garlic Bread, Wings, Pasta, etc.)
- **Footer** – Links, contact info (111 366 466), and disclaimer

## Tech Stack

- React 18
- Vite
- CSS (custom properties, flexbox, grid)

## Run locally

```bash
cd dominos-clone
npm install
npm run dev
```

Then open http://localhost:5173
